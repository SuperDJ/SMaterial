Chips

There are a couple types of chips that you can use. For now you can use the default, outlined, and input.

Chip

For the default chip all you need to use is the class .chip.

Chip

Outlined Chip

For the outlined chip you also need to use the class .chip--outlined The outlined chip can also contain a icon or image. These elemtents do not require any additional classes.

Chip

Input Chip

For the input chip you also need to use the class .chip--input The input can also contain a icon or image. These elemtns do not require any additional classes. The close action however does need the class .chip--input--close.
