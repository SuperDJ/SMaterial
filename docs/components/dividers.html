Dividers

Dividers can be used in either a horizontal or vertical way.

Horizontal

Dividers can be used horizontal. 

    
<div class="divider"></div>

<hr class="divider">

Vertical

Dividers can also be used vertical. 

    
<div class="divider divider--vertical"></div>

<hr class="divider divider--vertical">

Settings

Scss CSS Result
$divider-height --divider-height Change divider height
$divider-background --divider-background Change divider background
$divider-margin --divider-margin Change divider margin
