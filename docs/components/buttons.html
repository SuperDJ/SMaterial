Buttons

There are 3 types of buttons you can choose from which are: contained, outline and text.

Eacht button uses the .button class but from there they are different.

Text

A text button can simply be made by using the following class .button__text. If you would also like to add an icon, it is also needs the class .button__text--icon-left or .button__text--icon-right. This depends on where you would like to position the icon. 

    
<button class="button button--text">Send</button>

<button class="button button--text" disabled>Send</button>

<button class="button button--text button--text--icon-left">
    <i class="material-icons icon">send</i>
    Send
</button>

<button class="button button--text button--text--icon-right">
    Send
    <i class="material-icons icon">send</i>
</button>

<button class="button button--contained">Send</button>

<button class="button button--contained" disabled>Send</button>

    Send

<button class="button button--contained button--contained--icon-right">
